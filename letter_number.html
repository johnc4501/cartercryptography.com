The Letter-Number Cipher

Description: The letter-number cipher is a simple encryption method that turns alphabetical plaintext into numbers. The letter-number cipher does not have a key and instead always encrypts by the same standards everytime it is used. Letters are encrypted into the number position that the letter is in the alphabet. This means that since A is the first letter in the alphabet it encrypts to 1, B encrypts to 2, all the way up to Z which encrypts to 26. For example lets encrypt name, John. J is the 10th letter in the alphabet, O is the 15th, H is the 8th, and N is the 14th. So with the letter-number cipher John will encrypt to [10, 15, 8, 14].




Letter-Number Cipher Function Instructions:

  1. Open the file in PyCharm.
  2. Right-Click anywhere on the code window and a pop-up menu will open.
  3. Locate the "Run 'file name'" option and click it.
  4. Locate the 'Run' section at the bottom of the Pycharm Application. Here you will follow the input instructions as they appear and type your input when prompted to.
  5. Input your Plaintext. Hit Enter.
  6. Look for the printed Ciphertext Output.
  7. To encrypt again keep the Pycharm Application open and start again on step 2.
