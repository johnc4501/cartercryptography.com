The Reverse Cipher

Description: The reverse cipher is a very simple encryption cipher that takes a plaintext input and reverses the order of the values to create ciphertext. This Cipher works with alphabetical letters, numbers, and even special keys such as '!, ?, @, and #'. This is a very easy cipher to decrypt however when combined with others ciphers, can strengthen the security of your ciphertext.

For example my name, 'John Carter', run through the reverse cipher will output to the ciphertext: retraC nhoJ




Reverse Cipher Function Instructions:

  1. Open the file in PyCharm.
  2. Right-Click anywhere on the code window and a pop-up menu will open.
  3. Locate the "Run 'file name'" option and click it.
  4. Locate the 'Run' section at the bottom of the Pycharm Application. Here you will follow the input instructions as they appear and type your input when prompted to.
  5. Input your Plaintext. Hit Enter.
  6. Look for the printed Ciphertext Output.
  7. To encrypt again keep the Pycharm Application open and start again on step 2.
