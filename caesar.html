The Caesar Cipher

Description: The Caesar Cipher is a famous version of the shift cipher by substituting a particular alpahbetical letter with another further along in the alphabet by a set amount of characters. For example if you set your shift to 2, A would encrypt two letters to the right in the alphabet and appear as C in the ciphertext, B would encrypt to D, and so on. To decipher you take the original key, in this case a shift to the right of 2 and reverse it by shifting left 2 or mathematically shifting -2, so a ciphertext C would shift backwards to a plaintext A and so on.


Formula: The mathematical formula for the caesar cipher is: e = (x + n) % 26. This forumla allows a shift to occur while always remaing between the values 0-25, a-z, respectively.

  • Variable e = value of the encrypted letter resulted from the Caesar shift
  • Variable x = value of plaintext letter
  • Variable n = number of numeric shifts to the right x must go in order to achieve the desired Caesar shift, (Presuming n is added to x. Use (x-n) to shift to the left or to decrypt a previous right shift.)





Caesar Shift Cipher Function Instructions:

  1. Open the file in PyCharm.
  2. Right-Click anywhere on the code window and a pop-up menu will open.
  3. Locate the "Run 'file name'" option and click it.
  4. Locate the 'Run' section at the bottom of the Pycharm Application. Here you will follow the input instructions as they appear and type your input when prompted to.
  5. Input your Plaintext. Hit Enter.
  6. Create your Caesar Shift Key by inputing a positive integer to shift right and a negative integer to shift left. Hit Enter. (You can also decrypt a message by shifting the opposite amount of what it was encrypted by)
  7. Look for the printed Ciphertext Output.
  8. To encrypt again keep the Pycharm Application open and start again on step 2.
