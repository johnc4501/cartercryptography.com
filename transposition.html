The Columnar Transposition Cipher

Description: The Transposition Cipher has a number of different versions. The version included in this Toolset is the Columnar Transposition Cipher. In this cipher the plaintext is reordered by the parameters of a key. A key is created by selecting an order of integers. The plaintext is rewritten from left to right and top to bottom below those integers using the length of the integers as the amount of columns. Example: plaintext = GEORGIA, key = 312

312
GEO
RGI
A

For Encryption the sequence of characters is reordered by the using the lowest value integer's column of characters as the beginning of the ciphertext. Then the next lowest value integer's column of characters is added to the ciphertext, and so on until the highest value integer's column of characters is finished.

Ciphertext for the example above will read as such (Column 1:EG) + (column 2: OI) + (column 3: GRA). Final Ciphertext reads as EGOIGRA




Columnar Transposition Cipher Function Instructions:

  1. Open the file in PyCharm.
  2. Right-Click anywhere on the code window and a pop-up menu will open.
  3. Locate the "Run 'file name'" option and click it.
  4. Locate the 'Run' section at the bottom of the Pycharm Application. Here you will follow the input instructions as they appear and type your input when prompted to.
  5. Input your Plaintext. Hit Enter.
  6. Create your key by inputing an integer sequence.
  7. Look for the printed Ciphertext Output.
  8. To encrypt again keep the Pycharm Application open and start again on step 2.
