The Simple Substitution Cipher


Description: The simple substitution cipher is a cryptography method in which the user encrypts a message by creating a key that replaces the plaintext character with a corresponding ciphertext character. The version of the simple substitution cipher on this website is the mono-alphabetic version. In this version the alphabet is rearranged and the position of the original alphabetic letter in a message is replaced by the rearranged letter that takes it numeric value space. For example if you use XYZ as your first three letters in the rearranged alphabet key, your plaintext A will be encrypted to appear as the ciphertext X, B will encrypt to Y, and C will encrypt to Z.

Simple Substitution Cipher Function Instructions:

  1. Open the file in PyCharm.
  2. Right-Click anywhere on the code window and a pop-up menu will open.
  3. Locate the "Run 'file name'" option and click it.
  4. Locate the 'Run' section at the bottom of the Pycharm Application. Here you will follow the input instructions as they appear and type your input when prompted to.
  5. Type E for encrypting Plaintext or D for decrypting Ciphertext. Hit Enter.
  6. Input your Plaintext or Ciphertext. Hit Enter.
  7. Create your key by rewriting the alphabet in the order in which you want to encrypt each letter. Make sure not to repeat or miss any character. Hit Enter. (If you want to later decrypt your message, keep this key and reuse it when you select D for Decrypt.)
  8. Look for the printed Ciphertext or Plaintext Output.
  9. To Encrypt or Decrypt again keep the Pycharm Application open and start again on step 2.
