The Vertical Cipher

Description: The vertical cipher is characterized by the rearranging of a string of characters into a set amount of rows, which are then organized by the first letter being first letter of the first row, the second letter being the first letter of the second row, and so on for how many rows there are. When you have reached the end of the first letter for each row, the message is then further encrypted by repeating the same process but this time for the second of the first row, until the string has run out of characters.

For example if the message "ENCRYPTION" was encrypted into 3 rows of characters, the Ciphertext would appear as such, [ERTN] [NYI] [CPO]. Structured vertically the ciphertext would appear as

[ERTN]
[NYI]
[CPO]

If you read the first column top to bottom, then the subsequent columns, also top to bottom, the ciphertext would be legible almost as plaintext would. Technically, the vertical cipher does not have to appear vertically however when the rows are alligned vertically by reading the rows top to bottom and left to right, you can read the message with relative ease.




Vertical Cipher Function Instructions:

  1. Open the file in PyCharm.
  2. Right-Click anywhere on the code window and a pop-up menu will open.
  3. Locate the "Run 'file name'" option and click it.
  4. Locate the 'Run' section at the bottom of the Pycharm Application. Here you will follow the input instructions as they appear and type your input when prompted to.
  5. Input your Plaintext. Hit Enter.
  6. Input your key as an intger, specifying the amount of rows the ciphertext will be grouped in. Hit Enter.
  7. Look for the printed Ciphertext Output.
  8. To encrypt again keep the Pycharm Application open and start again on step 2.
