Welcome to my Webiste. This purpose of this webiste is to give access to my Encryption Cipher Toolset and to educate on how each cipher works and what they do. The Encryption Cipher Toolset is a set of Graphical User Interfaces created with Python and the Python Add-on Tkinter and include interactable encryption ciphers that accept a user's Plaintext input and outputs encrpyted ciphertext. The user creates their own key for each cipher.


Before you can use the Encryption Cipher Tools you will need the following

  • The newest version of Python if you'd like to open the files via the Python command line. OR
  • A Python Editor such as PyCharm if you'd like an easy way to open and run the files. The instructions for each Cipher will use this method.
  • Download the two zip files listed below and unzip them.
  • Click to Download All Cipher GUIs Python files (*Buttons Do not Work*)
  • Click to Download All Cipher Function Python files

The Ciphers that you will be able to work with through this webiste are:

  • The Simple Substitution Cipher
  • The Transposition Cipher
  • The Caesar Cipher
  • The Vertical Cipher
  • The Autokey Cipher
  • The Letter-Number Cipher
  • The Reverse Cipher
