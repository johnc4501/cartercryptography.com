"Build your Own Custom Cipher" Example

Description:

For a description of the custom cipher template please see the link to that page in the navigation above. The formula I chose for this example uses my birthday, 6/11/1999, as the numbers involved. I created a formula that shifts the ASCII unicode values of the letters (6 multiplied by 11) times to the right, and then shifts another 19 times to the right, and then shifts 99 spaces to the left. The formula for this shift is ( (6*11) + 19 - 99 ). This produces the value -14 which means the plaintext characters will shift 14 spaces to the left in the alphabet. Since we only need to change the 'custom_Shift_Formula = round()' line of code, the code in the example function python file is written as 'custom_Shift_Formula = round((6*11) + 19 - 99)'.




Example created with Cipher Template Function Instructions:

  1. Open the file in PyCharm.
  2. Right-Click anywhere on the code window and a pop-up menu will open.
  3. Locate the "Run 'file name'" option and click it.
  4. Locate the 'Run' section at the bottom of the Pycharm Application. Here you will follow the input instructions as they appear and type your input when prompted to.
  5. Input your Plaintext. Hit Enter.
  6. Look for the printed Ciphertext Output.
  7. To encrypt again keep the Pycharm Application open and start again on step 2.
